Welcome to Cerisse's documentation!
===================================

**Cerisse**  is a Large Eddy Simulation for Compressible Turbulent Flows with applications 
to combustion, high-speed flows. It uses the Block-Structured AMR Software Framework 
 `AMREX <https://amrex-codes.github.io/>`_
and offers a *simple* and *intuitive* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.
